# KO — Бизнес-аналитик | Сайт-визитка

Продающий лендинг-визитка бизнес-аналитика **Омуралиева Кутмана**.  
Сайт позиционирует услуги через формулу: **проблема клиента → решение → результат → доверие → CTA**.

🔗 **[Открыть сайт](https://omuralievk.github.io/KutEmpResume/)**

---

## Стек технологий

| Технология | Версия | Назначение |
|---|---|---|
| React | 19.2 | UI-компоненты, SPA |
| Vite | 5.x | Сборка и HMR |
| Tailwind CSS | 3.4 | Утилитарная стилизация |
| Font Awesome | 6.4 | Иконки (CDN) |
| Google Fonts | — | Inter + JetBrains Mono |
| Google Apps Script | — | Приём заявок и аналитика посещений |

## Возможности

- **Мультиязычность** — переключение RU / EN / KG с сохранением в `localStorage`
- **Тёмная / Светлая тема** — переключатель в навигации, тема сохраняется
- **Адаптивность** — полная поддержка мобильных устройств
- **Интерактивный фон** — Canvas-анимация частиц с реакцией на курсор
- **3D-карточки** — TiltCard компонент с эффектом наклона при наведении
- **Модальные окна** — детали услуг, карьерный роадмап с фото компаний
- **Аккордеоны** — болевые точки, FAQ
- **Форма обратной связи** — отправка заявок через Google Apps Script
- **Аналитика посещений** — сбор геоданных и отправка в Google Sheets
- **Lightbox** — полноэкранный просмотр фотографии
- **Анимации при скролле** — IntersectionObserver, staggered появление элементов

## Секции лендинга

1. **Hero** — фото, заголовок, trust-points, CTA
2. **Pain Points** — боли клиента (аккордеон)
3. **About Me** — фото + имя, карточка, модалка с карьерой / стеком / образованием
4. **Services** — 3 услуги с модальными деталями
5. **Deliverables** — конкретные артефакты на выходе
6. **Why Trust** — аргументы доверия
7. **Roadmap** — 10 этапов работы, zigzag-сетка с анимацией
8. **Pricing** — 4 тарифа
9. **FAQ** — аккордеон
10. **Contact** — форма + каналы связи
11. **Bottom CTA** — финальный призыв к действию

## Запуск

```bash
# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Деплой

Сайт деплоится на **GitHub Pages** из ветки `main` → папка `dist/`.  
Base URL настроен в `vite.config.js`:

```js
base: '/KutEmpResume/'
```
